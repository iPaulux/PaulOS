# PaulOS 2031

Portfolio interactif façon Windows 95, conçu pour un oral d'examen DA Senior.

**Live →** [paul2031.netlify.app](https://paul2031.netlify.app/)

---

## Concept

Une interface faux-OS inspirée de Windows 95/98 qui sert de support de présentation pour un entretien de Directeur Artistique Senior chez Grinto. Chaque icône du bureau ouvre une fenêtre dédiée à un aspect du profil.

## Contenu

| Fenêtre | Contenu |
|---|---|
| Mon métier | Fiche de poste + missions |
| Mon environnement | Journée type + setup créatif |
| Mes réalisations | Portfolio projets (lightbox) |
| Mes compétences | Barres de compétences + formation |
| Mes défis | Erreurs résolues style dialogue Win95 |

## Stack

HTML · CSS · JavaScript — zéro dépendance externe.

## Easter egg

Un lien discret se cache en bas à droite du bureau.
